# 博客与笔记

在这个博客上似乎已经刷了一周的绿瓷砖了（目移），
填了点有的没的小功能，
虽然大多数都是从别人那里抄的，
但暂且记录一下，
从别人那里借鉴的内容我会附上原链接。

作为博客折腾指北的序，
所以我不妨从头讲起，
讲点没人想听的故事。

## 在一切开始之前

最开始是为什么有搭建一个网站的想法呢？

我也记不得了，
虽然我手中有几个 SNS 账号，
也有一个公众号，
但是这种账号知道的熟人太多了，
自己放不开，
也不是很常去写点东西。

而且怎么说呢，
自认为自己算是掌控欲很强的人，
尤其对于一些有关于自己的事情而言。
总归还是不喜欢那种数据啊服务啊全都在别人手中，
尤其还是某些企业手中。

导出、自定义也都很不方便，
但也一时没想到什么好的替代品，
也就只能先凑合过着，
好在当时表达欲也没有那么严重。

## ThinkPad & Apache2

大概是 54 年夏天，我还在原专业的时候，
当时手头主用的电脑还是那台 `ThinkPad` 小黑
（有关于这台大概产于 43 年的传奇电脑的故事我感觉以后还能写个两三篇文章）。
因为电脑性能不足甚至难以运行 Win10 ，
我就给它装了一个 Debian 12 先用着。

在了解了一些 IP 地址和域名之后，
我抱着试试玩的心态在浏览器栏输入了 `127.0.0.1` ，
想看看能蹦出来什么东西。
其实当时本来觉得什么都不会有，
但不知是系统预装还是有软件依赖，
总之系统上安装了 apache2 ，
而且还是默认启动的。
于是我便看到了 apache2 的默认页面。

因为页面是全英文的，
而偏偏我英文还~~本当下手~~，
在查阅了些相关资料后，
我大概 ~~（完全没有）~~ 知道了这个页面是什么。
起码知道了这个页面 html 文件的位置，
于是我把我当周 C 语言的作业放了上去（传给舍友），
替换原有的默认 html 界面。
这样在连接宿舍 WiFi 之后访问电脑 IP
（啊对对，局域网是这样的），
就能看到这份作业。

这大概就是我某种意义上的**第一个网站**，
虽然它既不能在互联**网**上访问，
看上去也不像是一个**站**点。

我是~~杀软~~二次元，
看到这个简陋的界面，
第一反应就是找点法子寻思美化一下。
于是在一番搜索后我找到了 Jekyll ，
听说 Github 就是使用 Jekyll 生成页面的就打算试试，
不过可惜的是当时我似乎因为没有安装成功放弃了。

::github{repo="jekyll/jekyll"}

于是在另一番搜索后我又找到了 Hexo ，
果断安装，
阅读文档和教程的时候发现可以免费部署到 Github Pages 上，
这下不仅美化解决了，
也顺带解决了公网访问的问题。
那段时间基本也就是尝试各个主题，
ParticleX、ShokaX什么的都试了一遍
（你问我怎么找的？搜索栏里填一下 `anime` 就出来了）
最后因为弟弟君的原因选了 gal 主题。
不过主题试了很多，
但是基本没有内容产出。

## RaspberryPi & Wordpress

过了没多久，
我突然反应过来一个问题：

> 我博客扔到 Github 上，那我 Apache 怎么办？

本着技能点点都点了，
也不能荒着不用，
况且 hexo 尽管能让我有很大一部分自主掌控权，
但总归也是托管在了其他服务器上。

然后脑袋一抽，
决定还是把自己机子当作服务器的好。
在经过了多次比较后，
选择了广泛使用的 Wordpress ，
然后本着打死都改不了的死宅二次元的属性选择了 Sakurairo 主题。

然后又回到了那个经典问题：
我该怎么随时随地访问。

受限于局域网，
我还是只能在连到宿舍wifi后（并且电脑也连着）才能访问。
于是开始尝试弄个公网IP，
但是查了些资料发现公网IP正在收紧，
而且我也不是什么会和接线员啊客服啊交流的社交高手。
那么看看第二条路：内网穿透。
一搜，要钱，告辞。

就这样寻思了两天，
边查资料边想办法，
某天在河西吃饭的时候突然想起来。

> IPV4搞不到，我去搞IPV6啊，反正现在IPV6根服务器也没被淹

回宿舍登上路由器后台，
把IPV6开关打开，
然后出门试试看，
确实能通过IP访问了。

> 随时访问的需求就这样解决了……吗？
>
> 残念，并没有。

IPV6动不动就要更新一下，
每次电脑重新连到网就要记一下新的地址。

**而且IPV6那么长一串\[xyzw::qprs\]鬼才背的下来啊草**

那都到这个地步了，
不搞个域名实在是有些说不过去了。
按网上教程而言，
服务器这边用 ddns-go 动态更新到域名服务商那边就好了，
看上去也没多难。
于是随手就在阿里云上买了个`.top`，
正打算换上，
一看，
需要备案。

> 而且我也不是什么会和接线员啊客服啊交流的社交高手。
>
> ——我自己于没几个自然段之前

啊草一想到备案要交涉社恐就立刻发作了。
然后光速查找国外域名注册服务，
查到了 Cloudflare ，
毕竟打 CodeForces 的时候没少被他家人机验证拦，
起码混了个眼熟。
注册帐号后挑最便宜的 `.uk`
（US$ 5.x一年应该是，
折过来也就CN¥ 40不到，
我难道不想要`.moe`吗（哭）
注册了 `chiyoyuki.uk` 域名。

~~能用PayPal支付针不戳啊针不戳，刚好之前为了赞助Udon弄的PayPal还能派上用场~~

> 于是随时访问的需求就这样解决了……吗？
>
> 残念，还是没有。

We noticed that
服务器在线时间刚好严格等于我电脑连到IPV6网路的时间。
这就很 hot or bad.
所以最新需求就是我需要一台
能 24h 开机的、
能放宿舍的、
能连网的、
能跑起来Wordpress的
**电脑**。
